# 키워드 필터 — 구글시트로 관리 (실무자용)

뷰어의 **키워드 칩**을 코드/깃 없이 **구글시트만 편집**해서 바꿉니다.
시트를 고치고 뷰어를 새로고침하면 반영됩니다. (스토리유형·트로프 등 나머지 칩은 데이터에서 자동 생성 — 손댈 필요 없음)

## 시트 만들기 (최초 1회)

1. 구글시트에서 탭 하나 만들고 이름을 예: **`키워드필터`**
2. **1행에 헤더** 4개를 그대로 입력:

   | 라벨 | 키워드 | 검색범위 | 사용 |
   |---|---|---|---|

3. 2행부터 필터를 한 줄씩 추가. 예시:

   | 라벨 | 키워드 | 검색범위 | 사용 |
   |---|---|---|---|
   | 재벌/CEO | `재벌\|CEO\|billionaire\|회장\|보스` | | |
   | 질투 | `질투\|연적\|jealous` | | |
   | 계약/위장결혼 | `계약\|위장\|가짜\|contract` | | |
   | 임시로 끄기 | `테스트` | | N |

### 각 칸 규칙
- **라벨**: 칩에 표시될 이름 (자유롭게)
- **키워드**: 이 중 **하나라도** 대사·설명·훅에 있으면 걸림. **여러 개는 `|`(막대기)로 구분** — 쉼표 아님!
- **검색범위** *(비워도 됨)*: 기본은 설명·대사·훅 전체를 검색. 특정 범위만 원할 때만 `desc|script.line|hook_desc` 중 골라 `|`로.
- **사용** *(비워도 됨)*: 비우면 켜짐. `N` 넣으면 그 줄은 잠시 끔(삭제 안 하고 숨기기).

## 뷰어에 연결 (최초 1회, 담당자/개발자)

1. 시트 → **파일 → 공유 → 웹에 게시**
2. 게시 대상 = **`키워드필터` 탭**, 형식 = **쉼표로 구분된 값(.csv)** → **게시**
3. 나온 URL을 복사 (형식: `https://docs.google.com/spreadsheets/d/e/…/pub?gid=…&single=true&output=csv`)
4. 그 URL을 `reference/filters.json`의 `keyword_sheet_csv_url`에 붙이고 재배포
   (`python3 reference/build_viewer.py --deploy --password 'tain1234'` → commit → push)

> 이 연결은 처음 한 번만. 이후 실무자는 **시트만 편집**하면 됩니다 — 재배포 불필요, 새로고침하면 반영.

## 동작·한계
- 시트 로드 실패(게시 해제·오타 등) 시 뷰어는 `filters.json`에 박아둔 **예비 키워드**로 자동 폴백 — 화면이 비지 않음.
- 시트는 "웹에 게시"라 URL만 알면 공개로 읽힘 — 키워드 목록이라 민감정보 아님. (뷰어 본문은 별도로 비번 게이트로 보호됨)
- 반영 지연: 구글 게시 CSV는 캐시가 있어 편집 후 수 분 뒤 반영될 수 있음.
